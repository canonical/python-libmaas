Releasing a new version of python-libmaas

  1. Clean and test:

    make clean
make test

  2. If you didn’t make clean just now, do it! Without it the PyPI uploads may be built incorrectly.

  3. Bump version in setup.py, merge to master.

  4. Tag master:

    git tag --sign ${version} --message "Release ${version}."
git push origin --tags

  5. Build and push docs to GitHub:

    make docs-to-github

  6. Build and push source and wheel to PyPI:

    make upload
